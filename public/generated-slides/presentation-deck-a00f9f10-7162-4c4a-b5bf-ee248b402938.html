Mastra 是面向 TypeScript/Node.js 的 AI Agent 开发框架

原生 JS 生态

完美契合 Node.js 开发习惯，无需切换语言栈，降低心智负担。

轻量易集成

提供 Agent、Tool、Workflow 等原语，轻松嵌入现有后端服务。

开发体验佳

内置 Memory 等能力，快速构建具备上下文状态的智能应用。

核心概念：Agent 与 Tool
模块三：核心概念与基础实战 / P4

Agent (智能体)

具备角色设定与模型交互能力的实体，负责推理与决策。

defineAgent({ name: 'Assistant', instructions: '...' })
➔

Tool (工具)

Agent 可调用的函数，如 API 查询、计算等扩展能力。

defineTool({ name: 'getWeather', execute: async () => {...} })
* Memory 与 Workflow 将在后续模块中详细展开

为 Agent 添加自定义工具
模块三：核心概念与基础实战 / P6
const weatherTool = defineTool({ name: 'getWeather', execute: async (city) => { return `晴, 25°C`; } }); const agent = new Agent({ name: 'Assistant', model: 'openai/gpt-4o-mini', tools: [weatherTool] // 挂载工具 });

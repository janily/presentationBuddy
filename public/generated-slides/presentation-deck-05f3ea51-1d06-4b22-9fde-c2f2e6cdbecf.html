Mastra 开发框架入门

面向 TypeScript 开发者的 AI Agent 构建指南

Audience: TS/Node.js Devs
什么是 Mastra？

原生 TypeScript/Node.js 优先的 Agent 框架

类型安全、轻量、可组合（Agents + Tools）

定位：比通用 LLM 库更高层，比重型框架更灵活

核心概念矩阵

01Agent具备指令与模型的执行单元
02Tools赋予 Agent 外部调用能力
03Workflows多步状态编排
04Memory会话上下文持久化
05RAG检索增强生成集成

环境准备与安装

# Node.js >= 18 (支持 ESM)
npm install @mastra/core
# 可选 AI SDK
npm install @ai-sdk/openai

定义第一个 Agent

import { Agent } from '@mastra/core';
import { openai } from '@ai-sdk/openai';

const agent = new Agent({
  name: 'assistant',
  instructions: 'You are a helpful assistant.',
  model: openai('gpt-4o'),
});

为 Agent 添加工具

import { createTool } from '@mastra/core';
import { z } from 'zod';

const weatherTool = createTool({
  id: 'weather',
  inputSchema: z.object({ city: z.string() }),
  outputSchema: z.object({ temp: z.number() }),
  execute: async ({ city }) => ({ temp: 20 }),
});
const agent = new Agent({
  name: 'assistant',
  instructions: '...',
  model: openai('gpt-4o'),
  tools: [weatherTool],
});

执行与流式响应

const response = await agent.stream({
  messages: [{ role: 'user', content: 'Hello' }],
});

for await (const chunk of response.textStream) {
  process.stdout.write(chunk);
}

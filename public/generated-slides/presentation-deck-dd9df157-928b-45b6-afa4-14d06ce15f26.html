认识 Mastra：TS 原生的 Agent 框架

  • 定位：基于 TypeScript 原生的 AI 应用开发框架（底层依托 Vercel AI SDK）。
  • 核心模块：Agent（智能体）、Tools（工具）、Memory（记忆）、Workflow（工作流）、Evals（评估）。
  • 为何选择：类型安全（Zod 深度集成）、无缝接入 TS 生态、轻量且适合全栈部署。

环境准备与项目初始化

# 环境要求：Node.js >= 18 $ node -v v18.18.0 # 初始化项目（交互式脚手架） $ npm create mastra@latest # 或手动安装核心依赖 $ npm i @mastra/core @ai-sdk/openai # 配置环境变量 .env $ echo "OPENAI_API_KEY=sk-..." > .env
# 生成的项目目录树 my-mastra-app/ ├── src/ │ ├── mastra/ │ │ ├── agents/ # 智能体定义 │ │ ├── tools/ # 自定义工具 │ │ ├── workflows/ # 工作流 │ │ └── index.ts # Mastra 实例入口 ├── .env └── package.json

核心实战：创建你的第一个 Agent

// src/mastra/agents/index.ts import { Agent } from '@mastra/core/agent'; import { openai } from '@ai-sdk/openai'; export const myAgent = new Agent({ name: 'Helper', instructions: '你是一个乐于助人的中文 AI 助手。', model: openai('gpt-4o') }); // 运行实例 const response = await myAgent.generate('介绍一下你自己'); console.log(response.text);
> 运行 ts-node src/index.ts AI: 你好！我是一个由 Mastra 框架驱动的 AI 助手。 我可以帮助你解答问题、编写代码，并执行各种任务。 有什么我可以帮你的吗？

赋予能力：为 Agent 定义 Tools

import { createTool } from '@mastra/core/tools'; import { z } from 'zod'; export const weatherTool = createTool({ id: 'get-weather', description: '获取指定城市的当前天气', inputSchema: z.object({ city: z.string().describe('城市名称') }), execute: async ({ context }) => { const { city } = context; // 模拟 API 请求 return { city, temp: 25, condition: 'Sunny' }; } });
User: 北京今天天气怎么样？ [Tool Call] get-weather({ city: '北京' }) [Tool Result] { temp: 25, condition: 'Sunny' } AI: 北京今天天气晴朗，气温 25 度，适合外出。

进阶机制：Memory 与 Workflow

Memory 记忆

引入 @mastra/memory 实现多轮对话上下文保留。

import { Memory } from '@mastra/memory'; const memory = new Memory(); const agent = new Agent({ name: 'ChatBot', memory, // 注入记忆 model: openai('gpt-4o') });

Workflow 编排

使用 createWorkflow 和 createStep 将确定性任务流水线化。

const step1 = createStep({ id: 'fetch', execute: ... }); const step2 = createStep({ id: 'parse', execute: ... }); const wf = createWorkflow({ id: 'pipeline' }) .then(step1) .then(step2) .commit();
场景对比： 自由对话用 Agent + Memory，固定业务管线用 Workflow。

综合演练：构建一个本地文件查询助手

// src/mastra/index.ts import { Mastra } from '@mastra/core'; import { myAgent } from './agents'; import { fileTool } from './tools'; import { memory } from './memory'; export const mastra = new Mastra({ agents: { myAgent }, tools: { fileTool }, memory });
# 启动本地 Playground 或 API 服务 $ mastra dev > Server running on http://localhost:4111 User: 帮我读取 src/config.json 的内容 [Tool Call] fileTool({ path: 'src/config.json' }) AI: 这是 config.json 的内容：{ "debug": true, "port": 3000 }

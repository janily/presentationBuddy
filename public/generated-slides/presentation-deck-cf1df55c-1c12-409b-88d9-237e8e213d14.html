什么是 Mastra？

  • 定义：Mastra 是一个原生面向 TypeScript/Node.js 的 Agent 开发框架。
  • 核心优势：类型安全（Zod 集成）、与 JS 生态（Next.js/Vite）无缝衔接、声明式工作流。
  • 适用场景：构建具备推理能力和外部工具调用能力的 AI 应用。

简单工具调用 (Tool Calling)

1. 定义工具 (createTool)

import { createTool } from '@mastra/core';
import { z } from 'zod';

const getWeather = createTool({
  id: 'getWeather',
  description: '获取指定城市的天气',
  inputSchema: z.object({
    city: z.string()
  }),
  execute: async ({ city }) => {
    return `${city} 今天是晴天，25°C。`;
  }
});
➔

2. 挂载工具到 Agent

const agent = new Agent({
  name: '天气助手',
  instructions: '使用工具回答天气问题。',
  model: openai('gpt-4o'),
  tools: { getWeather } // 注入工具
});

实战：Agent 与工具协同

import { Agent, createTool } from '@mastra/core';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';

const getWeather = createTool({
  id: 'getWeather',
  description: '获取天气',
  inputSchema: z.object({ city: z.string() }),
  execute: async ({ city }) => `${city} 晴天`
});

const agent = new Agent({
  name: 'WeatherAgent',
  instructions: '回答天气问题',
  model: openai('gpt-4o'),
  tools: { getWeather }
});

const res = await agent.generate('北京天气如何？');
console.log(res.text);
> 运行脚本... [Agent] 识别意图：需要查询北京天气 [Tool] 调用 getWeather({ city: '北京' }) [Tool] 返回：北京 晴天 北京今天是晴天，适合外出。

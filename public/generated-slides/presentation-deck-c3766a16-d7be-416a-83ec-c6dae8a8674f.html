开发实战

快速开始：构建你的第一个 Agent

main.py
import asyncio
from claude_agent_sdk import Agent

async def main():
    # 定义系统指令
    agent = Agent(
        system_prompt="你是一个有帮助的AI助手。"
    )
    # 异步运行 Agent
    response = await agent.run("你好！")
    print(response)

if __name__ == "__main__":
    asyncio.run(main())
运行结果

工具调用（Tool Use）实战

定义本地函数并注册为 Tool，Agent 将自主触发。

tools.py
def get_weather(city: str) -> dict:
    return {"city": city, "temp": 25}

# 通过 schema 注册为 Tool
agent.register_tool(
    name="get_weather",
    func=get_weather
)

错误处理与调试最佳实践

error_handling.py
try:
    res = await agent.run(prompt)
except RateLimitError as e:
    print("触发限流，请重试")
except ToolExecutionError as e:
    print("工具执行失败")

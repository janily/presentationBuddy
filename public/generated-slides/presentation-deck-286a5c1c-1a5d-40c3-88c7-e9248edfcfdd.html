什么是 Mastra？

  • 基于 TypeScript 的 Agent 开发框架，类型安全
  • 原生封装 LLM 工具调用、流式输出与可观测性
  • 声明式定义 Agent / Tool / Workflow / Memory

环境初始化

  • 环境要求：Node.js ≥ 18
  • 使用官方脚手架秒级初始化

终端执行：

$ npm create mastra@latest
$ cd my-mastra-app && npm run dev

演讲者注释：实例化 Mastra 实例并配置模型与提示词

import { Mastra } from '@mastra/core';
export const mastra = new Mastra({
  agents: {
    helper: async () => {
      const { Agent } = await import('@mastra/core');
      return new Agent({ name: 'Helper', model: 'gpt-4o', instructions: 'You are a helpful dev assistant.' });
    }
  }
});

为 Agent 挂载工具

  • 使用 createTool 定义入参 schema 与执行逻辑
  • 将 tool 挂载至 Agent 的 tools 数组
const getTime = createTool({
  id: 'get-time',
  schema: z.object({}),
  execute: async () => new Date().toISOString()
});
// 挂载: new Agent({ tools: { getTime } })

记忆与状态管理

  • 通过 threadId 隔离不同用户/会话
  • 支持向量数据库或 KV 存储作为后端
  • 适用场景：多轮对话、用户偏好记忆、历史追溯

State is isolated by threadId

记忆集成代码

要点：在 Agent 配置中注入 Memory 实例

const agent = new Agent({
  name: 'Chat', model: 'gpt-4o',
  memory: new Memory({ options: { threadId: 'user-123' } })
});
await agent.generate('记住我的名字是 Tom', { threadId: 'user-123' });

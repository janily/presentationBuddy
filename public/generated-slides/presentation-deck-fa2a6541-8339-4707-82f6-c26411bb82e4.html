Agents 智能体

搭载 LLM 的动态决策核心，自主思考、规划并驱动工具链完成开放式任务。

Hands-on

实战代码：编写并运行你的第一个 AI 智能体

weather-agent.ts
import { z } from "zod";
import { createTool, Agent } from "@mastra/core";

// 1. 定义强类型天气工具
const weatherTool = createTool({
  id: "get-weather",
  inputSchema: z.object({ city: z.string() }),
  outputSchema: z.object({ tempC: z.number() }),
  execute: async ({ city }) => fetchWeather(city),
});

// 2. 创建 Weather Agent 并注入工具
const weatherAgent = new Agent({
  name: "weather-agent",
  instructions: "你是本地天气播报员",
  model: openai("gpt-4o-mini"),
  tools: { weatherTool },
});

// 3. 直接运行
const res = await weatherAgent.generate("上海今天天气如何？");
console.log(res.text);
Output
{ "city": "上海", "tempC": 27, "text": "上海今天多云，27°C，体感舒适。" }

Core Pillar · 02
核心支柱：Tools（定义强类型的自定义工具）
介绍 Tool 的概念、使用 Zod 定义参数 Schema，以及 Agent 如何智能调用工具。
// tools/weather.ts
export const weatherTool = createTool({
  id: 'get-weather',
  inputSchema: z.object({
    city: z.string(),
  }),
  outputSchema: z.object({ temp: z.number() }),
  execute: async ({ city }) => {
    return fetchWeather(city);
  }
});
01
自定义强类型工具：createTool API 强制绑定 Zod Schema 校验输入输出
02
纯 TS 体验：参数与返回值拥有完美的编辑器自动补全与类型守卫
// 调用闭环

↓

↓
结果返回 outputSchema 校验后回传 Agent
